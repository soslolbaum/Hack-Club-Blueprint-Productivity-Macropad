# Hack-Club-Blueprint-Productivity-Macropad
A Macropad designed for improving productivity and workflow on Linux.
Can and will be used for working on projects, hobbies homework ect...
Multiple buttons connected to rotary encoders to switch their functionality.
Features like:
       Execute a script that blocks distracting apps for 30 minutes
       Toggle do not disturb mode
       Toggle WiFi/VPN/Firewall
       Control brightness
       Switch arround workspaces
       Open project applications like GitHub, Blender, ArduinoIDE...
       Backup predetermined folders using Git/GitHub
and hopefully even more features to come!
